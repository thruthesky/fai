# Questions for Practice 1

## Basic Dart Practice Questions

This practice set covers fundamental Dart programming concepts including variables, data types, string manipulation, user input, and basic mathematical operations.

### Practice Problems

1. **Print Your Name**
   Create a program that outputs your name to the console.

2. **String Quotes Practice**
   Write code displaying "Hello I am "John Doe"" and "Hello I'am "John Doe"" using both single and double quotation marks appropriately.

3. **Constant Integer Declaration**
   Declare an immutable integer variable with a value of 7.

4. **Simple Interest Calculator**
   Implement a program calculating simple interest using the formula: `(p * t * r) / 100`, where p is principal, t is time, and r is rate.

5. **Square Calculator with Input**
   Create a program accepting user input and computing the square of that number.

6. **Full Name Concatenation**
   Write code combining first name and last name from user input into a complete name.

7. **Quotient and Remainder**
   Develop a program finding both the quotient and remainder from dividing two integers.

8. **Number Swapping**
   Create logic to exchange the values of two numbers.

9. **Whitespace Removal**
   Write code that eliminates all spaces from a string.

10. **String to Integer Conversion**
    Convert string values to integer data types.

11. **Bill Splitting Calculator**
    Implement a program dividing total bill amount by number of people: `(total) / (number of people)`.

12. **Travel Time Calculator**
    Calculate minutes needed to travel 25 km at 40 km/hour using: `(distance) / (speed)`.
