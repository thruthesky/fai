# Collections In Dart - Dart Tutorial

## Overview

This section covers Dart's collection types, which are fundamental data structures for organizing and managing groups of data.

## Topics Covered

The Collections In Dart module includes the following subjects:

1. **List in Dart** - Ordered, mutable collections of elements
2. **Set in Dart** - Unordered collections of unique values
3. **Map in Dart** - Key-value pair associations
4. **Where in Dart** - Filtering collections based on conditions
