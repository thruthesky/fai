# Dart Collections Practice Questions

## Question For Practice 4

Based on the Collections In Dart section, here are the eight practice exercises:

### Exercise 1: List of Names
"Create a list of names and print all names using list."

Implement a program that initializes a list containing multiple names and iterates through it to display each name.

### Exercise 2: Set of Fruits
"Create a set of fruits and print all fruits using loop."

Build a set data structure with fruit entries and use looping constructs to output all elements.

### Exercise 3: Expense Tracker
"Create a program thats reads list of expenses amount using user input and print total."

Develop an application that accepts expense amounts from user input, stores them in a collection, and calculates the sum.

### Exercise 4: Days of Week List
"Create an empty list of type string called days. Use the add method to add names of 7 days and print all days."

Initialize an empty string list named `days`, populate it with weekday names using the `add()` method, then display all entries.

### Exercise 5: Friends List with Filtering
"Add your 7 friend names to the list. Use where to find a name that starts with alphabet a."

Store seven friend names and apply the `where()` method to filter names beginning with the letter 'a'.

### Exercise 6: Map with Update
"Create a map with name, address, age, country keys and store values to it. Update country name to other country and print all keys and values."

Build a map with four keys, assign values, modify the country entry, and display complete key-value pairs.

### Exercise 7: Map Filtering
"Create a map with name, phone keys and store some values to it. Use where to find all keys that have length 4."

Construct a map and utilize `where()` to identify keys with exactly four characters.

### Exercise 8: To-Do Application
"Create a simple to-do application that allows user to add, remove, and view their task."

Develop an interactive application supporting task creation, deletion, and display functionality.
