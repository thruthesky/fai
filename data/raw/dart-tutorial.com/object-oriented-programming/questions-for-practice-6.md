# OOP Practice Questions for Dart

Based on the tutorial content, here are the seven exercises from "Questions for Practice 6":

## Exercise 1: Laptop Class
Develop a program creating a `Laptop` class with properties (id, name, ram). Instantiate three objects and display all details.

## Exercise 2: House Class with Constructor
Build a `House` class featuring properties (id, name, price) with a constructor. Create three objects, add them to a list, and display their information.

## Exercise 3: Gender Enum
"Write a dart program to create an enum class for gender [male, female, others] and print all values."

## Exercise 4: Animal and Cat Inheritance
Design an `Animal` class with properties (id, name, color), then extend it with a `Cat` class adding a `sound` property. Create and display a Cat object.

## Exercise 5: Camera with Encapsulation
Construct a `Camera` class with private properties (id, brand, color, price). Implement getters and setters to manage values across three objects.

## Exercise 6: Interface with Factory Constructor
"Create an interface called Bottle and add a method to it called open()." Implement a `CokeBottle` class, add a factory constructor to `Bottle` returning `CokeBottle` objects, and demonstrate calling `open()`.

## Exercise 7: Quiz Application
Develop a quiz application utilizing object-oriented principles allowing users to participate and view scores.
