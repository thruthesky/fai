# Community and Support

## Overview

The Dart community provides multiple channels for staying informed, contributing to the project, and connecting with other developers. The project maintains "an open and welcoming environment" and adheres to a code of conduct.

## Stay Informed

### Key Resources

- **Dart announce**: A low-traffic mailing list for "new releases, breaking changes, and other important news"
- **Dart blog**: Features insights from various Dart community members
- **@dart_lang**: Official X (Twitter) account for announcements
- **@dart.dev**: Official Bluesky account

## Contribute

Dart is open source with contribution guidelines available for the core SDK.

### Primary Repositories

- **Dart SDK repo**: Contains "the VM, JS and Wasm compilers, analysis, core libraries, and more"
- **Dart language repo**: Focuses on language design decisions
- **Dart website repo**: Source code for the official site, built with Jaspr
- **Dart code reviews**: High-traffic list of all core SDK code reviews

### Additional Projects

- **DartPad**: An online editor supporting console, web, and Flutter applications

## Join the Conversation

### Community Platforms

- **Stack Overflow**: Recommended for how-to questions
- **dart_community Discord**: Direct peer support channel
- **Dart on Reddit**: Community discussion forum
- **It's All Widgets Flutter Forum**: Asynchronous discussion space
- **Flutter communities**: Additional support resources

### Google Groups

- **General discussions**: Miscellaneous Dart topics
- **Dart analyzer**: Technical questions about the analyzer
