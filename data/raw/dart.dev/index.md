# Dart Programming Language - Overview

## Introduction

Dart is described as "an approachable, portable, and productive language for high-quality apps on any platform." The language is free and open source, supported by Google.

## Key Characteristics

### Approachable
Dart offers several developer-friendly features:

- **Modern Syntax**: A consistent, concise, and strongly typed programming language with familiar syntax
- **Safety Features**: Includes "sound null safety" and "collection if" operators
- **Advanced Patterns**: Supports pattern matching capabilities
- **Concurrency**: Features mature async-await functionality paired with isolate-based concurrency

### Portable and Fast

The language supports multiple compilation targets:

- **Native Compilation**: AOT-compile to native machine code for ARM, x64, and RISC-V architectures
- **Web Targets**: Compiles to both JavaScript and WebAssembly
- **Platforms**: Enables development for mobile, desktop, backend, and web applications

### Productive

Development experience improvements include:

- **Hot Reload**: Instantly view changes in running applications
- **Flexible Type System**: Rich static analysis with configurable tooling
- **Developer Tools**: Integrated profiling, logging, and debugging capabilities

## Platform Support

- **Web Platform**: Complete, mature, fast compilers for JavaScript and WebAssembly
- **Native Platform**: Direct compilation to machine code for instant startup
