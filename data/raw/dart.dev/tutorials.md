# Dart Tutorials Page

## Overview

The Dart tutorials page serves as a gateway to learning the Dart programming language and building applications across different platforms. The page is organized into three main sections designed to guide learners from fundamentals to advanced application development.

## Page Structure

### Learn the Basics

This section targets developers with foundational Dart knowledge who want to expand their skills. It recommends:

- **Asynchronous Programming: Streams** — "Use streams to manage sequences of data"
- **How to use packages** — "Organize and share code on the pub.dev site"

The introductory content suggests reviewing the language tour and asynchronous programming tutorials before progressing to iterables, streams, and package management.

### Build Server-Side Apps

This section covers tutorials for creating standalone applications:

- **Get Started** — Covers installation, creating simple apps, and compiling to native code
- **Write Command-Line Apps** — Features essential for CLI applications
- **Fetch Data from the Internet** — Using the HTTP package for web requests
- **Write HTTP Servers** — Implementing web servers in Dart

### Keep Learning

Additional resources include:

- Dart cheatsheet for quick language feature exploration
- Records and patterns codelab
- Iterable collections tutorial
- Flutter development resources

## Technical Details

The page includes Google Analytics and Tag Manager integration. It uses a file-tree CSS component for potential navigation elements and supports theme customization (light, dark, automatic modes).
